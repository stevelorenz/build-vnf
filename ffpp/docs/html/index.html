FFPP - Fast fast Packet Processing Library

This library is developed for fast prototyping and teaching high performance packet processing inside Linux containers. To be utilized low-level technologies including:

  • DPDK: Fast packet processing in user space.
  • XDP: Fast network data path in the Linux kernel.

Check the README page to get started.
